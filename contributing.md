# Contributing

We welcome contributions!

1. Fork the repo  
2. Create a branch:  
   ```bash
   git checkout -b feature/my-feature
   ```
3. Commit your changes:  
   ```bash
   git commit -m "Add new feature"
   ```
4. Push to your fork:  
   ```bash
   git push origin feature/my-feature
   ```
5. Open a Pull Request
